# 安哥拉外贸产品展示网站 - 完整开发规划

## 1. 项目概览

- **项目名称**: Angola B2B Product Showcase
- **项目类型**: WordPress 企业网站
- **核心目标**: 构建一个现代、动感、高性能的多语言（简体中文/繁体中文/葡萄牙语/英语）B2B产品展示平台，服务于安哥拉及全球客户。
- **管理后台**: 简体中文（固定）
- **前端展示**: 支持 4 种语言切换
- **技术栈**:
    - **核心平台**: WordPress
    - **主题**: 自定义主题 (基于 `_s` 或 `Hello Elementor`)
    - **主要动画库**: GSAP (GreenSock Animation Platform) + ScrollTrigger Plugin
    - **备用动画库**: AOS (Animate On Scroll) Library
    - **页面构建 (可选)**: Elementor Pro (用于快速布局)
    - **多语言插件**: WPML Pro 或 Polylang Pro
    - **自定义字段**: Advanced Custom Fields (ACF) Pro
    - **轮播组件**: Swiper.js (触摸友好)
    - **图片查看器**: PhotoSwipe (响应式Lightbox)
    - **图片优化**: WebP转换 + 响应式图片
    - **多端适配**: CSS Grid + Flexbox + 媒体查询

5.4. 多端适配完整方案

【核心理念】
移动优先 (Mobile First) + 渐进增强，确保在所有设备上都有优秀的用户体验。

### 5.4.1. 断点系统 (Breakpoints)

标准断点:
- 手机竖屏: 320px - 480px
- 手机横屏: 481px - 767px
- 平板竖屏: 768px - 1024px
- 平板横屏: 1025px - 1280px
- 桌面电脑: 1281px - 1920px
- 大屏幕: 1921px+

CSS媒体查询变量:
- --breakpoint-xs: 480px
- --breakpoint-sm: 768px
- --breakpoint-md: 1024px
- --breakpoint-lg: 1280px
- --breakpoint-xl: 1920px
