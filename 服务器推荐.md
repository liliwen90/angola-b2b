# 外贸网站服务器推荐方案

## 🖥️ 服务器基本配置要求

### 推荐配置（标准版）
适合中等流量的B2B网站：

- **CPU**: 2核心或以上  
- **内存**: 4GB RAM（最低2GB，推荐4GB）  
- **存储空间**: 50GB SSD起步（推荐100GB）  
- **带宽**: 不限流量或每月至少5TB流量  
- **操作系统**: Linux（Ubuntu 20.04/22.04 或 CentOS 7/8）

### 为什么需要这些配置？
- **2核CPU**：WordPress本身+4种语言+GSAP动画需要一定处理能力
- **4GB内存**：运行MySQL数据库+PHP+缓存，2GB太小会卡
- **100GB SSD**：产品图片（包括360度图片）、视频、备份都需要空间
- **大带宽**：图片和视频较多，流量消耗大

---

## 🌍 服务器位置选择（非常重要！）

### 针对安哥拉客户的最佳选择

由于主要客户在安哥拉（非洲），服务器位置至关重要：

**方案1：南非服务器（最推荐）**
- 物理距离最近
- 访问速度最快
- 推荐商家：AWS开普敦、Microsoft Azure南非

**方案2：欧洲服务器（次推荐）**
- 到安哥拉速度也不错（历史联系紧密）
- 推荐位置：葡萄牙、英国、德国
- 推荐商家：Vultr伦敦、DigitalOcean法兰克福

**方案3：美国/亚洲+CDN**
- 服务器在美国或香港/新加坡
- 必须配合CDN（Cloudflare）
- 价格较便宜但需要CDN补足速度

### 不推荐的选择
- ❌ 纯中国大陆服务器（需要备案，且到非洲慢）  
- ❌ 澳洲服务器（距离太远）

**优点**：按小时计费、可随时升级、界面简单  
**缺点**：客服一般

---

### 2. DigitalOcean（稳定首选）

**推荐原因**：稳定性好、文档丰富、开发者友好

**推荐套餐**：
- CPU: 2核心
- 内存: 4GB
- 存储: 80GB SSD
- 流量: 4TB/月
- 机房: 法兰克福或伦敦
- 价格: 约 $24/月（约160元人民币/月）

**优点**：非常稳定、一键安装WordPress、备份方便  
**缺点**：价格略高于Vultr

---

### 3. 阿里云国际版（中文支持）

**推荐原因**：有中文客服、支付方便、有香港/新加坡机房

**推荐套餐**：
- CPU: 2核心
- 内存: 4GB
- 存储: 40GB系统盘 + 100GB数据盘
- 流量: 按实际使用付费
- 机房: 香港或新加坡（配合CDN）
- 价格: 约 300-400元人民币/月

**优点**：中文客服、支付宝付款、熟悉的界面  
**缺点**：香港/新加坡到非洲需配CDN、价格较高

---

### 4. AWS (Amazon) 或 Azure（企业级）

**适合场景**：如果预算充足、需要最高稳定性

**推荐套餐**：
- AWS: t3.medium（2核4GB）在南非开普敦机房
- Azure: B2s（2核4GB）在南非约翰内斯堡
- 价格: 约 $40-60/月（约280-420元人民币/月）

**优点**：有南非机房（速度最优）、企业级稳定性  
**缺点**：价格较高、配置复杂

## 📊 推荐顺序

### 预算充足（500元/月以上）
**首选**：AWS南非 + Cloudflare CDN  
理由：速度最优、企业级服务、最接近客户

### 预算适中（300-500元/月）
**首选**：阿里云香港 + Cloudflare CDN  
理由：中文支持、支付方便、配合CDN速度可接受

### 预算有限（200元/月左右）
**首选**：Vultr伦敦 + Cloudflare CDN  
理由：性价比最高、到安哥拉速度尚可

## 🚀 必需的额外服务

### 1. CDN服务（内容分发网络）- 必须有！

**Cloudflare（强烈推荐）**
- **免费版已够用**（真的是免费！）
- 全球节点加速
- 自动图片优化
- 免费SSL证书
- 防DDoS攻击
- 设置简单

**使用Cloudflare后的效果**：
- 图片加载速度提升50%以上
- 视频流畅度大幅提升
- 全球访问速度均衡
- 节省服务器带宽

**替代方案**：
- BunnyCDN（$0.01/GB，便宜）
- KeyCDN（$0.04/GB）
- 七牛云CDN（如果客户也有中国访问需求）

---

### 2. SSL证书 - 必须有！

**推荐方案**：Let's Encrypt 免费证书
- 完全免费
- 自动续期
- 浏览器信任
- Cloudflare也提供免费SSL

**不需要花钱买证书**！

---

### 3. 域名注册

如果还没有域名，需要注册：

**推荐注册商**：
- **Namecheap**：价格实惠，$8-12/年
- **阿里云/腾讯云**：国内注册，支付宝，80-100元/年
- **Cloudflare Registrar**：价格透明，无溢价

**域名选择建议**：
- .com（最通用，首选）
- .co.ao（安哥拉本地域名，提升信任）
- .africa（非洲域名，较新）

---

### 4. 网站备份服务 - 强烈推荐

**方案1：服务器商自带备份**
- Vultr: $1/月自动备份
- DigitalOcean: $1.20/月自动备份
- 阿里云: 快照功能

**方案2：WordPress插件备份**
- UpdraftPlus免费版（备份到Google Drive/Dropbox）
- BlogVault（付费，$89/年，自动备份）

**推荐**：服务器商备份 + UpdraftPlus双保险

---

### 5. 邮件服务（询盘通知需要）

网站有询盘表单，需要发邮件：

**推荐方案**：
- **SendGrid**: 免费额度100封/天，够用
- **Mailgun**: 免费额度100封/天
- **阿里云企业邮箱**: 如需企业邮箱，约40元/月/账号

WordPress默认邮件功能不稳定，必须配置SMTP服务。

## 🛠️ 服务器必装软件

租用服务器后，需要安装：

1. **Web服务器**：Nginx 或 Apache
2. **数据库**：MySQL 8.0 或 MariaDB 10.6
3. **PHP**: 8.0 或 8.1版本
4. **SSL工具**：Certbot（自动申请Let's Encrypt）
5. **服务器管理面板**（可选）：
   - **宝塔面板**（推荐，中文界面，免费）
   - **Plesk**（付费，功能强大）
   - **cPanel**（付费，最流行）

**如果不懂Linux命令**，强烈建议装宝塔面板，可视化管理，就像操作Windows一样简单。

## ⚠️ 重要注意事项

### 1. 存储空间规划
- 每个产品平均10-20张图片
- 360度图片每个产品36-72张小图
- 预计100个产品需要 20-30GB
- 视频如果多需要更大空间
- **建议至少100GB存储**

### 2. 流量消耗
- 图片和视频网站流量消耗大
- 有CDN情况下，3TB/月流量够用
- 没有CDN，可能需要5TB+
- **务必选不限流量或大流量套餐**

### 3. 安全防护
- 必须装Wordfence等安全插件
- 定期更新WordPress和插件
- 强密码
- 定期备份
- Cloudflare自带DDoS防护

### 4. 性能优化
- 必须装缓存插件（WP Rocket或W3 Total Cache）
- 图片必须优化（ShortPixel插件）
- 开启Gzip压缩
- 使用CDN
- 数据库定期优化

### 5. 监控和维护
- 装UptimeRobot监控网站是否宕机（免费）
- 定期检查网站速度（GTmetrix、Pingdom）
- 每周检查一次备份是否正常
- 每月更新一次WordPress核心和插件
