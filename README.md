# Angola B2B Product Showcase

## 项目概述

现代化的多语言（简体中文/繁体中文/葡萄牙语/英语）B2B产品展示平台，专为安哥拉及全球客户打造。

## 技术栈

- **WordPress**: 6.4+
- **PHP**: 8.0+
- **主题**: 自定义主题（angola-b2b）
- **动画**: GSAP + ScrollTrigger
- **多语言**: WPML Pro
- **自定义字段**: ACF Pro
- **响应式**: Mobile-first设计

## 项目特性

- 4种语言支持（简体中文、繁体中文、葡萄牙语、英语）
- 管理后台固定为简体中文
- 产品360度旋转展示
- 高级动画效果
- 完整的多端适配（手机/平板/桌面）
- 询盘管理系统
- 产品翻译管理
